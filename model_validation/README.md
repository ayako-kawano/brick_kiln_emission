# Model vs Monitor PM2.5 Validation

Statistical validation of the AELMO/WRF d02 fine-PM model output against
Bangladesh DoE CAMS monitor observations (Oct 2022 – Sep 2023).

Run `python ../validate_model_vs_monitors.py` to regenerate all metrics and
plots (edit the path constants at the top of the script for your machine).
`paired_hourly.csv` (the full hourly paired table) is regenerable and is not
committed here.

See `VALIDATION_REPORT.md` for methods, results, and interpretation.
